bindInt

inline fun SeekBar.bindInt(range: ClosedRange<Int>, obs: MutableObservableProperty<Int>): Unit

Binds this SeekBar two way to the observable property. When the user picks a new value from the seek bar, the value of the observable property will change to the new value. When the value of the observable property changes, the seek bar will be adjusted accordingly.
