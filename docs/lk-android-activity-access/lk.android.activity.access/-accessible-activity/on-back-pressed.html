onBackPressed

open val onBackPressed: ArrayList<() -> Boolean>

Overrides ActivityAccess.onBackPressed

When the back button is pressed, the lambdas in the list are invoked in reverse order until one of the lambdas returns true, indicating that the button press has been handled.

It is suggested that one add a lambda to the top of the list upon creation of a view that needs its back button handled, and then to remove that lambda when the view goes away.
