Package description

Adds a couple of extension functions that should probably be in Anko, such as functions to create RecyclerViews with scrollbars.

Dependencies

This package depends upon:

  • AppCompatV7
  • Design
  • Anko.

Gradle Inclusion

Add the repository:

repositories {
    maven {
        url "https://dl.bintray.com/lightningkite/com.lightningkite.kotlin.android"
    }
}

Include the desired libraries:

dependencies {
    implementation "com.lightningkite.kotlin.android:anko-extensions:[version]"
}
