ActivityAccess

interface ActivityAccess

An interface for accessing activities in a decentralized way, where multiple listeners can listen to event like onPause, onResume, etc, and most importantly, can use prepareOnResult to set up lambdas to occur when a result code is received.

Created by joseph on 6/9/17.
